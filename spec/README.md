# Пакет для ИИ-агента: «Дашборд рассадки сотрудников»

Этот пакет содержит материалы, которые можно напрямую передать ИИ-агенту для генерации локального frontend-приложения.

## Файлы

1. `01_TZ_employee_seating_dashboard.md` — полное техническое задание.
2. `02_AI_AGENT_PROMPT.md` — готовый prompt для ИИ-агента.
3. `03_DATA_MODEL_EXAMPLE.json` — пример структуры данных проекта.
4. `04_ACCEPTANCE_CRITERIA.md` — критерии приемки.
5. `05_IMPLEMENTATION_NOTES.md` — технические ограничения и приоритеты реализации.
6. `00_FULL_AGENT_CONTEXT.md` — единый объединенный файл со всем контекстом.

## Как использовать

Рекомендуемый порядок:

1. Передать агенту `02_AI_AGENT_PROMPT.md`.
2. Прикрепить `01_TZ_employee_seating_dashboard.md` как полное ТЗ.
3. Прикрепить `03_DATA_MODEL_EXAMPLE.json` как ориентир по структуре данных.
4. Использовать `04_ACCEPTANCE_CRITERIA.md` для проверки результата.

## Ключевые ограничения

- Только HTML + CSS + Vanilla JavaScript.
- Backend отсутствует.
- Приложение работает offline.
- Запуск двойным кликом по `index.html`.
- Без CDN и внешних API-запросов.
- Локальные библиотеки допускаются только в папке `libs/`.
- Drag-and-drop и расчеты нельзя упрощать.
